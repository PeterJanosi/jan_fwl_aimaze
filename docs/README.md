# Dokumentáció — jan_fwl_aimaze

Ez a `docs/` mappa a projekt részletes, disztribúció-specifikus és implementációs jegyzeteit tartalmazza. A cél, hogy a beadandóhoz szükséges technikai információk, konfigurációk és algoritmiai döntések jól dokumentáltak legyenek magyar nyelven.

Fájlok:

- `rl_spec.md` — Állapot / Akció / Jutalmazás (State / Action / Reward) specifikációk.
- `simulation_setup.md` — Szimulációs környezet beállítása (Gazebo és Ignition, ROS 2 disztribúció-specifikus tippek).
- `training.md` — Tanítási folyamat, javasolt hyperparaméterek, checkpointing és logolás.

Bővebb tartalmak: minden fájl célja, hogy önállóan használható legyen mint jegyzet és mint belső dokumentáció a beadandóhoz.
